# Phase 0 Lesson 2: Git & Collaboration

## Goal

Learn how to save project progress with Git.

## Key Ideas

- Working directory: files you are editing now.
- Staging area: changes selected for the next snapshot.
- Commit: a saved snapshot in your local Git history.
- Remote: a copy of the repository on a service such as GitHub.

## Command Flow

```powershell
git status
git add phase0-lesson2-git.md
git commit -m "Add Git lesson notes"
```

## Notes

- `git status` shows what changed.
- `git add` chooses what goes into the next commit.
- `git commit` saves the chosen changes as a snapshot.
